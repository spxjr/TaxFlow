# TaxFlow
Full-stack PHP/MySQL tax CRM — client profiles, tax return tracking, task management, live search, reports, and a mobile-responsive dashboard. Built with vanilla PHP, PDO, and CSS.
TaxFlow CRM is a purpose-built client relationship manager for tax and accounting firms. Built with PHP, MySQL (PDO), and vanilla CSS — no frameworks, no build tools, just clean server-rendered pages that work.

Features
• Dashboard — live KPIs, filing season progress, activity feed, and work pipeline
• Clients — searchable grid/list view, client profiles with contact editing, document tracking, and invoice history
• Returns — tax return management across all form types (1040, 1120-S, 1065, 1120, 1041) with status tracking, preparer checklists, and K-1 shareholder splits
• Tasks — priority-grouped task board with due date intelligence, assignee tracking, and one-click completion
• Reports — revenue by month, filing completion donut charts, top clients, and full invoice tables with year filtering
• Live Search — global search with 220ms debounce, result caching, keyboard navigation, and highlighted matches
• Mobile-first — responsive layout with slide-in sidebar, bottom navigation bar, and mobile search drawer

Tech Stack
• Backend: PHP 8+, PDO/MySQL
• Frontend: Vanilla CSS (no framework), vanilla JS (no dependencies)
• Database: MySQL with 9 relational tables, foreign keys, and indexes
• Fonts: DM Serif Display, DM Mono, Instrument Sans via Google Fonts
